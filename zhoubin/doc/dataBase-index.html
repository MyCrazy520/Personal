MySql安装使用

MySql安装

						wget http://dev.mysql.com/get/mysql57-community-release-el7-8.noarch.rpm  #下载mysql源安装包

						yum localinstall mysql57-community-release-el7-8.noarch.rpm #安装mysql源

						yum repolist enabled | grep "mysql.*-community.*" #检查mysql源是否安装成功

						#修改 vim /etc/yum.repos.d/mysql-community.repo源 ，改变默认安装的mysql版本。比如要安装5.6版本，将5.7源的enabled=1改成enabled=0。然后再将5.6源的enabled=0改成enabled=1即可

						yum install mysql-community-server #安装MySQL

						systemctl start mysqld #启动MySQL服务

						systemctl enable mysqld #开机启动
						systemctl daemon-reload  #开机启动

						grep 'temporary password' /var/log/mysqld.log #修改root本地登录密码 1.查看mysql密码

						mysql -uroot -p #修改root本地登录密码 2.连接mysql

						ALTER USER 'root'@'localhost' IDENTIFIED BY 'Zhoubin77!'; #修改密码[注意：后面的分号一定要跟上]

						GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY 'Alax0518$%QvodteERb1940' WITH GRANT OPTION; #添加远程登录用户

mysql导入导出sql文件

导出数据和表结构
						/usr/local/mysql/bin/   mysqldump -uroot -p abc > abc.sql  #敲回车后会提示输入密码
导出表结构
						#/usr/local/mysql/bin/   mysqldump -uroot -p -d abc > abc.sql
导入数据库方法1
mysql>create database abc; #1、首先建空数据库 mysql>use abc; #2.导入数据库:选择数据库 mysql>set names utf8; #2.导入数据库:设置数据库编码 mysql>source /home/abc/abc.sql;#2.导入数据库:导入数据（注意sql文件的路径）
导入数据库方法2
#mysql -uabc_f -p abc < abc.sql
导入数据遇到的问题
  • 问题：set global max_allowed_packet=524288000;
  • 方法1：临时修改:
    • 							mysql>set global max_allowed_packet=524288000;
							mysql> show VARIABLES like '%max_allowed_packet%’; #查看是否修改了
  • 方法2：修改my.cnf
    • 							max_allowed_packet=256M
MySQL出现too many connections（1040）错误解决方法
  • 修改my.cnf,添加如下:
						max_connections=10000
mysql column which is not functionally dependent GROUP BY clause
  • 修改my.cnf,添加如下:
						STRICT_TRANS_TABLES,NO_ZERO_IN_DATE,NO_ZERO_DATE,ERROR_FOR_DIVISION_BY_ZERO,NO_AUTO_CREATE_USER,NO_ENGINE_SUBSTITUTION
